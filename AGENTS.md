## Charte éditoriale (français)

**Ne jamais utiliser le tiret cadratin (U+2014, `—`)** dans les textes du site (guides, FAQ, SEO, calculateurs, composants, messages utilisateur, commentaires de contenu visible).

Privilégier selon le contexte : virgule, deux-points, parenthèses, point, nouvelle phrase, ou tiret simple `-` uniquement s'il est justifié (ex. cellule vide de tableau, placeholder calculateur).

Reformuler si nécessaire pour garder un ton naturel et humain, sans modifier le sens.
